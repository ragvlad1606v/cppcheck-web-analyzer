# cppcheck-web-analyzer

Cppcheck Web Analyzer

A modern, web-based static analysis platform for C and C++ powered by Cppcheck.
This project allows users to upload source files and instantly receive security, bug, and code-quality analysis through an intuitive web interface with dark/light mode and a cyber-themed UI.

Features:

C & C++ static analysis using Cppcheck

File upload and live scan results

Dark / Light mode UI

Green cyber-themed interface

Web-based, no local installation required

Built for students, developers, and cybersecurity enthusiasts who want fast, accessible code analysis.
